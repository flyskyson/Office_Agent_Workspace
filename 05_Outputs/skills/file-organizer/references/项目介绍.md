## 🎯 项目功能

### 核心功能
1. **自动扫描** - 扫描桌面和下载文件夹的文件
2. **智能识别** - 识别文件类型（申请表、身份证、租赁合同、产权证明）
3. **自动归档** - 按日期和申请人分类整理
4. **统计报告** - 生成详细的整理报告

### 技术特点
- ✅ 配置和代码分离（通过config.json调整规则）
- ✅ 面向对象设计（类: FileOrganizer）
- ✅ 模块化结构（每个功能独立）
- ✅ 完善的注释和学习文档

## 💻 技术栈

**语言**: Python 3.x
**核心库**:
- `os` - 文件和文件夹操作
- `shutil` - 文件移动
- `json` - 配置文件管理
- `datetime` - 日期处理
