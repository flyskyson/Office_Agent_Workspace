## 🎯 项目进度追踪器 - 防止忘记项目进度

### 📁 工具位置
```
project_tracker.py
```

### 🚀 什么时候用?

#### ✅ 必须使用的场景:
1. **开始新项目前** - 先记录当前项目进度
2. **暂停项目时** - 记录做到哪了
3. **切换项目时** - 防止混乱
4. **忘记进度时** - 查看项目状态

#### 💡 使用频率建议:
- **每天早上**: `python project_tracker.py list`
- **暂停项目时**: `python project_tracker.py pause <项目名>`
- **完成任务时**: `python project_tracker.py update <项目名> <进度%>`

## 📋 常用命令速查

```bash
# 查看所有项目 (最重要!)
python project_tracker.py list

# 查看某个项目状态
python project_tracker.py status <项目名>

# 更新项目进度
python project_tracker.py update <项目名> <进度%> <当前任务>

# 暂停项目
python project_tracker.py pause <项目名> <原因>

# 恢复项目
python project_tracker.py resume <项目名>

# 开始新项目
python project_tracker.py start <项目名> <描述>
```

## 🎯 典型使用场景

### 场景1: 早上开始工作
```bash
python project_tracker.py list
→ 看到所有项目状态
→ 决定今天做哪个
```

### 场景2: 临时开发新项目
```bash
# 1. 记录当前进度
python project_tracker.py update file_organizer 80 "添加进度提示"

# 2. 暂停当前项目
python project_tracker.py pause file_organizer "临时开发backup_tool"

# 3. 开始新项目
python project_tracker.py start backup_tool

# 4. 新项目完成后,恢复原项目
python project_tracker.py resume file_organizer
```
