# 官方《个体工商户登记（备案）申请书》模板

## 下载来源

这些模板来自各地市场监督管理局官网，基于市场监管总局2022年3月1日实施的《市场主体登记文书规范》。

## 模板列表

- **广东版（2022标准）**: `official_application_form_guangdong.doc`
- **福建版**: `official_application_form_fujian.doc`
- **厦门版**: `official_application_form_xiamen.doc`


## 使用说明

1. 这些是官方模板，使用全国统一的格式
2. 建议对比广西政务服务平台的在线表格格式
3. 下一步将转换为Jinja2模板格式
