## 🛠️ 工程管理系统

### 项目管理方法论

采用**敏捷开发 + 看板**模式:

```
待办 (Backlog)
    ↓
开发中 (In Progress)
    ↓
测试中 (Testing)
    ↓
已完成 (Done)
```

### 看板工具

**简单工具**: Trello / GitHub Projects / Notion
**推荐**: GitHub Projects (因为代码已托管)

### 优先级决策框架

**价值/复杂度矩阵**:

```
高价值 + 低复杂度  →  🔥 立即做 (Do First)
高价值 + 高复杂度  →  📅 计划做 (Plan)
低价值 + 低复杂度  →  ⏰ 有空做 (Do Later)
低价值 + 高复杂度  →  ❌ 不做 (Don't Do)
```

### GLM Coding Plan 优势

您订阅的套餐带来的价值:

✅ **高额度**: 每月 120-2400 次调用 (Lite/Max)
✅ **高性能**: GLM-4.7 SOTA 模型
✅ **速度快**: 55+ Tokens/秒
✅ **无限制**: 不受网络限制
✅ **多功能**: 图像/视频理解 + 联网搜索

**最佳实践**:
- 充分利用高额度进行深度技术调研
