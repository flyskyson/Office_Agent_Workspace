# 🌅 今日启动器使用指南

## 什么是今日启动器？

**今日启动器**（Daily Launcher）是你每天打开工作区时的"晨间助手"，帮助你：

1. **快速了解工作区状态** - 健康检查、问题提醒
2. **回顾昨天的工作** - 显示最近修改的文件
3. **确定今天的任务** - 智能推荐待办事项
4. **直接开始工作** - 快速打开项目、工具

---

## 🚀 如何使用

### 方法1：通过启动菜单（推荐）

双击运行：
```
start_new_session.bat
```

然后选择选项 **1** - 🌅 今日启动器

### 方法2：直接运行

在命令行中运行：
```bash
python daily_launcher.py
```

或双击运行：
```
test_daily_launcher.bat
```

## 📊 启动器功能说明

### 1. 工作区健康检查

启动器会自动检查：

- ✅ **工作区索引状态** - 是否需要重新扫描
- ⚠️  **项目文档** - 哪些项目缺少README
- 🔵 **Git状态** - 是否有未提交的更改
- 📁 **文件状态** - 工作区是否整洁

**状态图标**：
- ✅ 良好 - 一切正常
- ⚠️  需要注意 - 有小问题但不影响工作
- ❌ 存在问题 - 需要立即处理

### 2. 最近24小时活动

显示你最近修改的文件，包括：

- 📁 文件路径
- ⏰ 修改时间（多久前）
- 📊 文件大小
- 🚀 所属项目

这帮你快速找回昨天的上下文！

### 3. 今日推荐任务

基于工作区状态智能推荐：

- 🔴 **高优先级** - 继续昨天的工作
- 🟡 **中优先级** - 补充文档、整理代码
- 🟢 **低优先级** - Git提交、清理缓存

### 4. 快速操作菜单

一键执行常用操作：

1. **刷新工作区索引** - 运行 `workspace_scanner.py`
2. **生成详细报告** - 运行 `workspace_report.py`
3. **清理工作区** - 运行 `workspace_cleaner.py`
4. **打开学习日志** - 记录今天的学习
5. **查看所有项目** - 浏览活跃项目
6. **打开VSCode** - 直接开始编码
0. **退出** - 结束启动器

## 🔧 高级用法

### 自定义启动器

编辑 `daily_launcher.py` 可以自定义：

- 修改欢迎语
- 添加更多健康检查项
- 调整推荐任务逻辑
- 添加新的快速操作

### 添加到开机启动

1. 创建快捷方式到 `daily_launcher.py`
2. 将快捷方式放到 `shell:startup` 文件夹
3. 每天开机自动运行启动器

### 集成到其他工具

启动器可以作为模块导入：

```python
from daily_launcher import DailyLauncher

launcher = DailyLauncher()
health = launcher.check_workspace_health()
recent = launcher.get_recent_files(hours=24)
```

## ❓ 常见问题

### Q: 启动器显示"无法找到工作区索引"？

**A**: 运行一次 `workspace_scanner.py` 生成索引：
```bash
python workspace_scanner.py
```

### Q: 显示的时间不准确？

**A**: 确保系统时间正确，或者最近修改的文件在24小时外。

### Q: 推荐任务不合适？

**A**: 推荐基于最近活动和工作区状态。你可以自由选择要做的事情！

### Q: 如何退出启动器？

**A**: 选择 `0` 或按 `Ctrl+C`
