# Smart Tools - 智能工具集

**项目类型**: Python 实用工具集合
**创建日期**: 2026-01-16
**状态**: 活跃

---

## 📂 包含工具

### 📰 smart_news_assistant.py
智能新闻监控助手
- 功能：记住用户兴趣，智能匹配相关新闻
- 支持平台：知乎、微博、B站、抖音、快手
- 特点：长期/短期兴趣管理，自动学习关键词

### 🚀 workflow_starter.py
工作流启动器
- 功能：快速启动预定义工作流模板
- 支持：自动执行、交互式选择、批量处理

### 📝 markdown_exporter.py
Markdown导出工具
- 功能：将记忆系统内容导出为Markdown格式
- 输出：结构化文档，便于阅读和归档

---

## 🚀 使用方式

```bash
# 智能新闻助手
python smart_news_assistant.py

# 工作流启动器
python workflow_starter.py

# Markdown导出
python markdown_exporter.py
```

## 📋 依赖

- `feedparser` - RSS订阅解析
- `requests` - HTTP请求
- `markdown` - Markdown格式化
