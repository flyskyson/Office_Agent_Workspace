## 🎯 最简单的方案（推荐）

### ⭐ 方案1: 桌面一键启动

#### 步骤1: 找到桌面快捷方式

```
桌面上的图标:
📱 Workspace Butler
```

#### 步骤2: 双击运行

```
双击 "Workspace Butler.lnk"
  ↓
自动执行:
  1. 扫描工作区
  2. 查看项目进度
  3. 生成管家报告
  4. 显示激活成功
```

#### 步骤3: 打开AI对话

```
打开 Claude Code (或VS Code)
  ↓
复制这段话发送给我:
```

```markdown
# 🏠 管家模式激活！

早上好！我已激活工作区管家。

请读取:
- 06_Learning_Journal/AI_MEMORY.md
- 06_Learning_Journal/workspace_memory/workspace_index_latest.json
- 06_Learning_Journal/workspace_memory/project_progress.json

然后告诉我:
1. 我的项目进度如何？
2. 今天推荐做什么？
3. 有什么需要注意的？
```

**就这么简单！** 🎉

## ⚡ 快捷启动技巧

### 技巧1: 创建VS Code代码片段

**文件**: VS Code → 用户代码片段

```json
{
  "Butler Mode": {
    "prefix": "butler",
    "body": [
      "# 🏠 管家模式激活！",
      "",
      "早上好！我已运行启动脚本。",
      "",
      "请帮我:",
      "1. 读取 06_Learning_Journal/AI_MEMORY.md",
      "2. 运行 python 智能管家项目查询.py list",
      "3. 总结我的项目状态",
      "4. 推荐今天的工作",
      "",
      "谢谢管家！"
    ],
    "description": "激活工作区管家模式"
  }
}
```

**使用**: 在VS Code中输入 `butler` → 自动展开

### 技巧2: 保存为常用提示词

在Claude Code中保存为"技能"(Skill):

**名称**: `管家激活`
**内容**: 激活话术模板

### 技巧3: 桌面提醒便签

在桌面创建文本文件 `管家激活话术.txt`:

```
# 每天激活管家的话术:

管家！

请查询项目状态并给出今日推荐。
```
