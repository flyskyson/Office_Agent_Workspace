# 工作区三大工具快速启动指南

## 🎯 一键启动命令

### 1. 超级管家 - 查看工作区全貌
```bash
python 超级管家.py
```
**功能**：
- 显示工作区完整信息
- 列出所有活跃项目（5个活跃项目）
- 显示可用工具（39个Python脚本）
- 显示最近活动和报告

**当前状态**：
- ✅ market_supervision_agent（27个文件）
- ✅ memory_agent（7个文件）
- ✅ pdf_processor（772个文件）
- ✅ file_organizer（4个文件）
- ✅ my_first_agent（400个文件）

---

### 2. 今日启动器 - 每日开始工作
```bash
python daily_launcher.py
```
**功能**：
- 显示时间问候（早上/下午/晚上）
- 工作区健康检查
- 活跃项目列表
- 快速启动项目

**菜单选项**：
- `status` - 查看工作区状态
- `projects` - 查看活跃项目
- `tools` - 查看可用工具
- `notes` - 查看最近笔记
- `report` - 生成工作区报告

---

### 3. 超级文件管理中心 - 整理和维护
```bash
python file_manager_center.py
```
**可用命令**：

#### 文件管理
- `organize` - 整理文件（智能分类归档）
- `clean` - 清理缓存和临时文件
- `scan` - 扫描工作区全貌

#### 快照与备份
- `snapshot` - 创建完整快照
- `restore <文件名>` - 恢复文件

#### 报告与监控
- `report` - 生成健康报告
- `status` - 查看当前状态

#### 维护工具
- `maintenance` - 运行定期维护
- `check-git` - 检查Git状态

**启动文件管理中心.bat**：
