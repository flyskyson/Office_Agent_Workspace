# 工作区工具使用指南
# 超级管家、今日启动器、文件管理中心

## 🎯 三大核心工具

### 1. 超级管家 (超级管家.py)
一键获取工作区所有关键信息，让AI能立即回答关于你工作区的任何问题。

**使用方法**：
```bash
python 超级管家.py
```

**功能**：
- 📊 获取MCP服务器配置
- 📁 工作区索引状态
- 📂 项目列表摘要
- 🛠️ 工具脚本摘要
- 📝 最近活动
- 📚 笔记和文档摘要
- ✅ 数据新鲜度检查

**输出格式**：JSON格式的完整报告

---

### 2. 今日启动器 (daily_launcher.py)
每天打开工作区的第一站，是你的"晨间助手"。

**使用方法**：
```bash
python daily_launcher.py
```

**功能**：
1. 🌅 快速了解工作区状态
2. 📋 回顾昨天的工作
3. 🎯 确定今天的任务
4. 🚀 直接开始工作

**特色功能**：
- 显示时间问候（早上/下午/晚上）
- 工作区健康检查
- 活跃项目列表
- 待办事项提醒
- 快速启动命令

**菜单选项**：
- `status` - 查看工作区状态
- `projects` - 查看活跃项目
- `tools` - 查看可用工具
- `notes` - 查看最近笔记
- `report` - 生成工作区报告
- `help` - 显示帮助

---

### 3. 超级文件管理中心 (file_manager_center.py)
统一入口，集成所有文件管理功能。

**使用方法**：
```bash
python file_manager_center.py
```

**可用命令**：

#### 📁 文件管理
- `organize` - 整理文件（智能分类归档）
- `clean` - 清理缓存和临时文件
- `scan` - 扫描工作区全貌

#### 📸 快照与备份
- `snapshot` - 创建完整快照
- `restore` - 恢复文件（需要指定文件名）

#### 📊 报告与监控
- `report` - 生成健康报告
- `status` - 查看当前状态

#### 🔧 维护工具
- `maintenance` - 运行定期维护
- `check-git` - 检查Git状态

#### 🤖 智能功能
- `auto` - 智能推荐并执行
- `help` - 显示帮助信息

**示例**：
```bash
# 整理文件
python file_manager_center.py organize

# 创建快照
python file_manager_center.py snapshot

# 生成报告
python file_manager_center.py report

# 智能推荐
python file_manager_center.py auto
```

## 🚀 快速开始

### 场景1：每天开始工作
```bash
# 第一步：启动今日启动器
python daily_launcher.py

# 第二步：查看项目状态
# 选择 "projects" 或输入 "p"

# 第三步：开始工作
# 根据提示选择项目或工具
```

### 场景2：整理工作区
```bash
# 启动文件管理中心
python file_manager_center.py

# 整理文件
organize

# 清理临时文件
clean

# 创建快照
snapshot
```

### 场景3：查询工作区信息
```bash
# 运行超级管家
python 超级管家.py

# 查看输出的JSON报告
# 包含所有项目、工具、最近活动等信息
```

## 💡 使用技巧

### 技巧1：批处理文件
创建批处理文件快速启动：

**启动今日启动器.bat**：
```batch
@echo off
chcp 65001
cd /d C:\Users\flyskyson\Office_Agent_Workspace
python daily_launcher.py
pause
```

**启动文件管理中心.bat**：
```batch
@echo off
chcp 65001
cd /d C:\Users\flyskyson\Office_Agent_Workspace
python file_manager_center.py
pause
```

### 技巧2：定期维护
建议每周运行一次：
```bash
python file_manager_center.py maintenance
python file_manager_center.py snapshot
```

### 技巧3：快照管理
创建快照前先整理：
```bash
python file_manager_center.py organize
python file_manager_center.py snapshot
```
