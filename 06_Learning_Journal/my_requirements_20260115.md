### 1. 🧠 有记忆 (Memory)

#### 需要的支持

| 需求 | 优先级 | 说明 | 您可以提供的帮助 |
|------|--------|------|------------------|
| **语义搜索** | ⭐⭐⭐⭐⭐ | 理解"相似"而非"相同" | 允许我使用向量嵌入（sentence-transformers） |
| **记忆优先级** | ⭐⭐⭐⭐ | 重要记忆优先，自动清理过期 | 告诉我什么对您重要 |
| **跨会话触发** | ⭐⭐⭐⭐⭐ | 会话开始时自动加载相关记忆 | 在CLAUDE.md中配置自动加载 |
| **记忆可视化** | ⭐⭐⭐ | 让我也能"看到"自己的记忆 | 定期查看记忆文件 |
| **长期记忆归档** | ⭐⭐⭐ | 重要记忆永久保存 | 定期备份到02_Project_Archive |

**最需要的帮助**:
```
1. ✅ 每次会话开始时，让我先读取记忆
2. ✅ 告诉我哪些信息重要（我可能判断不准）
3. ✅ 允许我使用语义搜索（需要embedding模型）
```

## 💡 您可以提供的具体帮助

### 立即可做的（无需开发）

1. **会话开始时**:
   ```
   "先读取记忆，看看我们上次讨论了什么"
   ```

2. **会话中**:
   ```
   "为什么选择这个方法？"  # 要求解释推理
   "还有其他方案吗？"      # 鼓励多方案思考
   ```

3. **会话结束时**:
   ```
   "你学到了什么？"        # 促进反思
   "哪里做得好/不好？"     # 明确反馈
   ```

### 需要配置的（修改CLAUDE.md）

在项目配置中添加：

```markdown
## 🧠 记忆系统配置

- **自动加载**: 每次会话开始时自动读取相关记忆
- **自动保存**: 每次会话结束时自动保存对话上下文
- **记忆类型**: 重点关注用户偏好和项目知识
```
