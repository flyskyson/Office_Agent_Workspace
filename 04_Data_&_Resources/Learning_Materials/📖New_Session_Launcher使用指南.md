## 🎯 这是什么？

**New Session Launcher** 是您的**工作区快速启动菜单**，用于在新对话中快速恢复上下文和访问常用工具。

**启动方式**: 双击 `start_new_session.bat` 或桌面快捷方式 `New Session Launcher.lnk`

---

## 🚀 8个功能速查表

| 选项 | 功能 | 用途 |
|------|------|------|
| ⭐ **1** | 🌅 今日启动器 | 晨间工作区助手（推荐！）|
| **2** | 查看我的开发者档案 | 让AI了解您的背景 |
| **3** | 运行项目规划助手 | 智能推荐项目 |
| **4** | 打开学习日志记录工具 | 记录学习内容 |
| **5** | 查看自动化项目库 | 浏览18个自动化项目 |
| **6** | 查看快速开始提示词 | 9个场景的提示词模板 |
| **7** | 生成工作区状态报告 | 健康检查 |
| **0** | 退出 | 退出菜单 |

---

## 💡 常用场景快速指南

### 🌅 每天开始工作
```
1. 运行 start_new_session.bat
2. 选择 ⭐ 1 - 今日启动器
3. 查看推荐和待办
4. 开始工作！
```

### 🆕 重开AI对话
```
1. 运行 start_new_session.bat
2. 选择 2 - 查看我的开发者档案
3. 复制档案内容发给AI
或告诉AI: "请读取 06_Learning_Journal/AI_MEMORY.md"
```

### 🎯 需要新项目
```
1. 运行 start_new_session.bat
2. 选择 3 - 运行项目规划助手
   或
3. 选择 5 - 查看自动化项目库
4. 选择适合的项目开始
```

### 📝 记录学习内容
```
1. 运行 start_new_session.bat
2. 选择 4 - 打开学习日志记录工具
3. 记录今日学习
4. 保存日志
```

### 🔍 需要提示词模板
```
1. 运行 start_new_session.bat
2. 选择 6 - 查看快速开始提示词
3. 找到合适的场景模板
4. 复制并修改使用
```

## 🛠️ 三个启动工具对比

| 工具 | 文件 | 用途 | 何时使用 |
|------|------|------|----------|
| 🏠 管家模式 | butler_mode.bat | 激活管家模式 | 需要AI全面了解工作区 |
| 🚀 启动菜单 | start_new_session.bat | 快速访问功能 | 日常使用最频繁 |
| 🏦 文件管理 | run_file_manager_center.bat | 文件管理中心 | 整理、清理工作区 |

## ⚡ 快捷方式

桌面上有3个快捷方式（如果运行过 `create_shortcut.ps1`）：

1. **Workspace Butler.lnk** - 工作区管家模式
2. **File Manager Center.lnk** - 文件管理中心
3. **New Session Launcher.lnk** - 新对话启动菜单 ⭐
