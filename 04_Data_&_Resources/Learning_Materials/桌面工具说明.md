# 工作区工具快速启动指南

## ✅ 已完成！

我已经为你创建了以下批处理启动文件：

### 📁 批处理文件（在工作区根目录）

1. **启动今日启动器.bat**
   - 每天开始工作的第一站
   - 查看工作区状态和项目列表
   - 直接启动项目工具

2. **启动文件管理中心.bat**
   - 统一管理所有文件
   - 整理、清理、快照功能
   - 生成工作区报告

3. **启动市场监管智能体.bat**
   - 直接启动Word申请表填充工具
   - 快速生成个体工商户申请书

## 🚀 使用方法

### 方法1：双击批处理文件（最简单）
直接双击 `.bat` 文件即可启动对应工具！

### 方法2：从命令行运行
```bash
# 启动今日启动器
.\启动今日启动器.bat

# 启动文件管理中心
.\启动文件管理中心.bat

# 启动市场监管智能体
.\启动市场监管智能体.bat
```

## 🎯 推荐工作流程

### 每天开始工作：
```
1. 双击 "启动今日启动器.bat"
2. 查看项目状态
3. 选择要处理的项目
4. 开始工作
```

### 整理文件时：
```
1. 双击 "启动文件管理中心.bat"
2. 输入 organize
3. 等待整理完成
```

### 生成申请表：
```
1. 双击 "启动市场监管智能体.bat"
2. 填写信息
3. 生成Word文档
```

## 💡 提示

- 所有批处理文件都设置了正确的编码（UTF-8）
- 每个工具运行结束后会暂停，方便查看输出
- 可以随时按 Ctrl+C 退出工具
