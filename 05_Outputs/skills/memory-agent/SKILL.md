# 记忆助手

**类型**: 智能体项目
**技术栈**: Streamlit, ChromaDB, sentence-transformers
**位置**: `01_Active_Projects/memory_agent`

## 技能概述

基于向量数据库的语义记忆存储和检索系统

## 核心功能
- **语义记忆存储**
- **向量搜索**
- **间隔复习**
- **记忆统计**

## 快速开始

### 启动服务
```bash
cd 01_Active_Projects/memory_agent
python ui/app.py
```
