## 🔧 故障排除

### 问题1：编码错误
**症状**：程序崩溃，显示Unicode编码错误
**解决方案**：
1. 使用文本生成器 `application_generator.py`
2. 或者运行快速测试 `quick_test_app.py`

### 问题2：Word模板问题
**症状**：Word文件无法打开或内容乱码
**解决方案**：
1. 用Word打开模板文件
2. 删除所有特殊符号（✓、☑等）
3. 只保留纯文本和表格
4. 另存为新文件

### 问题3：依赖包缺失
**症状**：ImportError: No module named 'docx'
**解决方案**：
```bash
pip install python-docx
```

## ❓ 常见问题

**Q: 为什么Word模板有编码问题？**
A: Word文件中的特殊符号（如复选框）在Python处理时可能产生编码冲突。

**Q: 文本生成器和Word工具有什么区别？**
A: 文本生成器生成纯文本文件，完全可用；Word工具生成.docx文件，但有编码问题。

**Q: 如何批量处理多个客户？**
A: 准备JSON数据文件，使用批量生成功能。

**Q: 生成的申请表可以直接使用吗？**
A: 是的，生成的文本申请表可以直接打印或转换为PDF使用。
