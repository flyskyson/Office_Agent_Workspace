## 你对我说的第一句话

当开始新对话时，你只需要说：

```
你好，帮我看看工作区状态
```

或

```
继续我们上次的工作
```

或

```
我想做 [某件事]
```

## 我会做什么

1. **读取上述文件**（自动）
2. **了解当前状态**
3. **等待你的具体指示**

## 你的角色

- **提问者**: 告诉我你想做什么
- **决策者**: 告诉我你的优先级
- **使用者**: 使用我创建的工具

## 我的角色

- **记忆者**: 记住工作区的所有变化
- **管理者**: 协调各个工具和系统
- **实施者**: 根据你的要求执行任务
- **解释者**: 当你不懂时，清楚解释

## 具体例子

### 你说
```
帮我升级申请书生成工具
```

### 我会做
1. 读取演进日志，了解当前版本
2. 备份现有代码
3. 实施升级
4. 记录变化
5. 告诉你改了什么

### 你说
```
工作区现在是什么状态？
```

### 我会说
1. 有哪些工具，什么版本
2. 最近做了什么升级
3. 下一步计划是什么

### 你说
```
我想要 [某个功能]
```

### 我会做
1. 理解你的需求
2. 检查是否已有类似功能
3. 如果没有，规划实施
4. 如果有，告诉你如何使用

## 当你不懂的时候

直接问：

- "这个是做什么用的？"
- "为什么要这样做？"
- "怎么使用这个功能？"
- "这个和那个有什么区别？"

我会用简单易懂的方式解释。

### 我期望你
- ✅ 告诉我你的目标
- ✅ 在不懂的时候提问
- ✅ 告诉我优先级
