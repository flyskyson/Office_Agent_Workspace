# 新对话快速开始指南

> 当你重开对话时，按照这个流程快速恢复上下文并继续合作

---

## 🚀 一键启动（推荐）

### Windows用户
双击运行：
```
start_new_session.bat
```

这个脚本会提供菜单让你：
1. 查看开发者档案
2. 运行项目规划助手
3. 打开学习日志工具
4. 查看项目库和提示词
5. 生成工作区报告

---

## 📋 方式1: 完整快速开始流程

### 步骤1: 发送标准开场白

将以下内容发送给新AI助手：

```
你好！我重开了一个新对话。

请先阅读我的开发者档案：
文件位置：06_Learning_Journal/AI_MEMORY.md

然后告诉我：
1. 我的技能水平和学习目标
2. 推荐接下来可以做的项目

我的背景概要：
- 我是公务员，主要做办公自动化和网页自动化
- 目前是编程初学者，通过项目学习
- 这个工作区有完整的项目记录和学习日志
```

### 步骤2: 让AI运行项目规划助手

```
请运行项目规划助手帮我分析：

python 06_Learning_Journal/project_planner.py

或者运行批处理文件：
start_new_session.bat

根据我的档案和当前进度，推荐适合的项目。
```

### 步骤3: 开始项目

AI助手会：
1. 分析你的AI_MEMORY.md
2. 推荐适合的项目
3. 帮助你开始开发

### 生成工作区报告
```bash

```

或者双击：

## 💬 与AI助手沟通技巧

### 好的提问方式
- ✅ "请先阅读AI_MEMORY.md了解我的背景"
- ✅ "基于我的技能水平，推荐适合的项目"
- ✅ "请解释为什么这样做，教我原理"
- ✅ "帮我记录这次的学习内容"

### 避免的方式
- ❌ "直接给我代码"（学不到东西）
- ❌ "帮我完成所有功能"（应该结对编程）
- ❌ 不提供上下文（AI无法针对性帮助）

## 🆘 遇到问题

### 工具无法运行
1. 检查Python是否安装：`python --version`
2. 检查工作目录：应该在 Office_Agent_Workspace/
3. 查看错误信息，告诉AI助手

### 文件找不到
1. 确认在正确的工作区根目录
2. 运行 `dir` 或 `ls` 查看文件列表
3. 运行 `start_new_session.bat` 查看选项

### AI不了解你
1. 明确告诉AI读取 AI_MEMORY.md
2. 提供具体的文件路径
3. 说明当前需求和学习目标
