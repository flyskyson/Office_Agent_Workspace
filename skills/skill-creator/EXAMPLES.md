# Skill Creator - 使用案例

## 案例 1: 从零创建文件备份技能

### 场景
用户想要一个自动备份文件的技能，但不知道如何编写 SKILL.md 文件。

### 完整对话流程

```
用户: 我有个想法，想做一个自动备份文件的技能

AI: # 🚀 Skill Creator 已激活

    好的想法！让我帮您创建这个技能。

    ## 📋 需求确认
    请确认以下信息：

    1. **技能名称**: `file-backup`
    2. **功能描述**: 自动备份指定目录到目标位置，支持增量备份和压缩
    3. **触发关键词**: `备份文件` `自动备份` `文件备份` `backup`
    4. **分类**: automation（自动化工具）

    确认无误吗？

用户: 确认

AI: ✅ 正在生成技能框架...

    [调用 skill_creator.py]

    ✅ 技能创建成功: skills/file-backup/
       主文件: skills/file-backup/SKILL.md
       附加文档: EXAMPLES.md, CONFIG.md, TROUBLESHOOTING.md

    💡 现在让我帮您填充核心内容...

    ## 📝 执行步骤设计

    建议的执行步骤：
    1. **检查源目录**: 验证源路径存在且可访问
    2. **创建备份目录**: 按日期创建备份文件夹
    3. **复制文件**: 支持增量复制（仅复制修改过的文件）
    4. **压缩备份**: 可选 ZIP 压缩
    5. **生成报告**: 记录备份文件列表和大小

    ## 💬 使用场景示例

    场景 1: 日常备份
    ```
    用户: 备份我的文档

    AI: 好的，正在备份...
        源: C:/Users/xxx/Documents
        目标: D:/Backups/Documents_20260114/

        ✅ 已备份 156 个文件 (234 MB)
    ```

    场景 2: 增量备份
    ```
    用户: 增量备份项目目录

    AI: 正在检查变更...
        ✅ 3 个新文件
        ✅ 2 个修改的文件
        ⏭️  跳过 150 个未变更文件

        ✅ 增量备份完成 (节省时间: 95%)
    ```

    ✅ 技能创建完成！您可以:
    - 说"备份我的文档"来测试
    - 编辑 skills/file-backup/SKILL.md 继续完善
```

### 预期结果

```
skills/file-backup/
├── SKILL.md               ✅ 包含完整执行步骤和场景
├── EXAMPLES.md            ✅ 详细的备份案例
├── CONFIG.md              ✅ 配置说明（源目录、目标目录等）
└── TROUBLESHOOTING.md     ✅ 常见问题（权限错误、磁盘空间不足等）
```

## 案例 2: 创建截图整理技能

### 场景
用户桌面截图太多，想要自动按日期/应用分类。

### 完整对话流程

```
用户: 我想做个整理截图的技能

AI: # 🚀 Skill Creator 已激活

    好的！让我帮您创建。

    ## 📋 需求确认
    请告诉我：

    1. **技能名称**: 建议 `screenshot-organizer`，可以吗？
    2. **整理规则**: 想按什么方式分类？
       - 按日期？（如 2024-01-14/）
       - 按应用？（如 WeChat/、Chrome/）
       - 按分辨率？（如 1920x1080/）
    3. **截图来源**: 哪些目录？
    4. **目标位置**: 整理到哪里？

用户: 按日期和应用，来源是桌面和下载文件夹，整理到 Pictures/Screenshots/

AI: 明白了！正在创建...

    ✅ 技能名称: `screenshot-organizer`
    ✅ 整理规则: 按日期 → 按应用
    ✅ 结构: Pictures/Screenshots/2024-01-14/WeChat/

    [生成框架...]

    ## 📝 执行步骤设计

    建议步骤：
    1. **扫描源目录**: 查找所有截图文件（PNG/JPG）
    2. **识别元数据**: 从文件名提取日期和应用信息
    3. **创建目录结构**: 按日期/应用创建文件夹
    4. **移动文件**: 保持原文件名，移动到对应目录
    5. **生成报告**: 列出整理的文件数量

    ## 💬 使用场景示例

    场景 1: 整理桌面截图
    ```
    用户: 帮我整理桌面的截图

    AI: 正在扫描桌面...
        找到 45 个截图文件

        正在整理...
        ✅ WeChat 截图: 12 个 → 2024-01-14/WeChat/
        ✅ Chrome 截图: 8 个 → 2024-01-14/Chrome/
        ✅ 其他截图: 25 个 → 2024-01-14/Other/

        ✅ 整理完成！桌面已清空
    ```

    场景 2: 定期整理
    ```
    用户: 每周整理一次下载文件夹的截图

    AI: 好的，我可以帮您设置自动提醒。
       或者您可以直接说"整理截图"，我会自动处理。
    ```

    ✅ 技能创建完成！
```

### 预期结果

```
skills/screenshot-organizer/
├── SKILL.md
├── EXAMPLES.md
├── CONFIG.md
└── TROUBLESHOOTING.md
```

## 案例 3: 批量创建相关技能

### 场景
用户正在构建一个图像处理工具集，需要创建多个相关技能。

### 完整对话流程

```
用户: 我需要创建 3 个图像处理技能

AI: 好的！批量创建模式已激活。

    请告诉我这 3 个技能的名称和功能：

    技能 1: ?
    技能 2: ?
    技能 3: ?

用户:
    1. image-compressor - 图片压缩
    2. image-resizer - 图片缩放
    3. image-converter - 格式转换

AI: 明白了！正在批量创建...

    ## 📦 技能 1: image-compressor
    ✅ 分类: automation
    ✅ 触发词: `压缩图片` `图片压缩` `减小图片大小`
    ✅ 创建成功

    ## 📦 技能 2: image-resizer
    ✅ 分类: automation
    ✅ 触发词: `缩放图片` `调整大小` `图片缩放`
    ✅ 创建成功

    ## 📦 技能 3: image-converter
    ✅ 分类: automation
    ✅ 触发词: `转换格式` `图片转换` `格式转换`
    ✅ 创建成功

    📊 批量创建完成！

    💡 后续建议:
    1. 统一验证: python skill_creator.py validate
    2. 共享工具函数: 创建 00_Agent_Library/image_utils.py
    3. 协调工作流: 创建 image-processor 总技能

    需要我帮您做这些吗？
```

### 预期结果

```

├── image-compressor/
│   └── SKILL.md
├── image-resizer/
│   └── SKILL.md
```
