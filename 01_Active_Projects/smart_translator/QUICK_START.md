## 🚀 立即使用（3 种方式）

### 方式 1: 浏览器书签（最简单）⭐⭐⭐⭐⭐

**步骤**：

1. **打开书签工具页面**
   ```
   双击: 00_Agent_Library\99_Scripts_Tools\启动翻译书签工具.bat
   ```
   或直接打开：[browser_tools.html](browser_tools.html)

2. **选择推荐的翻译引擎**
   - **Immersive Translate**（最推荐）- 双语对照、PDF 翻译
   - **DeepL for Chrome** - 最高翻译质量
   - **微软必应翻译** - 稳定可用

3. **复制书签代码**
   - 点击页面上的代码块
   - 会自动复制到剪贴板

4. **创建浏览器书签**
   - 按 `Ctrl+Shift+B` 显示书签栏
   - 右键书签栏 → "添加页面"
   - 名称：输入 "翻译"
   - URL：粘贴复制的代码
   - 保存

5. **使用书签翻译**
   - 在任意网页选中文字
   - 点击翻译书签
   - 自动打开翻译结果

### 方式 2: Chrome 扩展（推荐）⭐⭐⭐⭐⭐

#### Immersive Translate（强烈推荐）

**安装**：
1. 访问 [Chrome 网上应用店](https://chromewebstore.google.com/detail/immersive-translate-trans/bpoadfkcbjbfhfodiogcnhhhpibjhbnh)
2. 点击"添加到 Chrome"
3. 安装完成

**配置**：
1. 点击工具栏的 Immersive Translate 图标
2. 选择翻译引擎（推荐 DeepL 或微软）
3. 设置默认翻译语言为中文

**使用**：
- **快捷键**: `Alt + T` 翻译当前页面
- **双语对照**: 同时显示原文和译文
- **PDF 翻译**: 直接翻译 PDF 文件

#### DeepL for Chrome

**安装**：
1. 访问 [Chrome 网上应用店](https://chromewebstore.google.com/detail/deepl-for-chrome/clneakkikoojmpfofhlppjmbmfeodoje)
2. 添加到 Chrome

**特点**：
- 最高翻译质量
- 快速翻译
- 支持文档翻译
